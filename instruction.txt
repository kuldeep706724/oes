Installation Requirement

Step 1
Install jdk version 8 or above

Step 2
Install NetBeans version 8 or above from here:
https://netbeans.apache.org/download/index.html

Step3
Install Apache Tomcat from here:
https://tomcat.apache.org/download-80.cgi

Step4
Install MySQL server from here:
https://dev.mysql.com/downloads/installer/
download MySQL jar file from here:
https://dev.mysql.com/downloads/connector/j/

Step5
Open MySQL Command line and run these commands:-

create database kuldeepdb;
